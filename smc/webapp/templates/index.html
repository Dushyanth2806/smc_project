BOS / CHoCH Predictor

Train a model on Smart-Money structure breaks, then forecast the next bar.

1. Train from TradingView

Pulls live OHLCV history from TradingView (via tvDatafeed), labels every bar with the rule-based BOS/CHoCH engine, and trains a classifier to predict the next bar's event.

2. Train from an uploaded CSV

Required columns (case-insensitive): open, high, low, close, volume. A time/date column is recommended. Extra columns (e.g. prev_close) are ignored — prev-close and other derived features are computed automatically.

Event type: Structure: Model: Extra:

Every arrow points from the broken pivot to the bar that broke it. Color = direction (green = bullish break, red = bearish break) — not BOS vs CHoCH. Arrowhead + label = event type: BOS (solid arrowhead, continues the trend) vs CHoCH (hollow arrowhead, reverses the trend). A green CHoCH is normal — it means price just broke upward while the trend had been bearish. "Swing" = long (50-bar) pivot lookback, major structure, rare. "Internal" = short (5-bar) lookback, minor structure, frequent — both are real, rule-based BOS/CHoCH, just drawn smaller. Every confirmed swing pivot is labeled HH (higher high) / LL (lower low) / LH (lower high) / HL (higher low), toggled with "Swing (major)". A later BOS always breaks a prior HH or LL (trend continuation); a later CHoCH always breaks a prior LH or HL (trend reversal) — the arrow label shows which, e.g. "BOS (HH)". IDM (Inducement, amber) is not in the original indicator — it's a heuristic we added: a minor structure break swept the "wrong" way shortly before a real BOS/CHoCH fires the other way. OrderFlow is an approximation (no tick/bid-ask data exists here) derived from candle shape + relative volume — treat it as a rough proxy, not real order flow. Order Blocks and Fair Value Gaps are no longer shown on this chart.
